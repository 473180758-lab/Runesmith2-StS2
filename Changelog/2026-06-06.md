## Runesmith 2 2026-06-06 Changelog (v0.4.0)

### Visuals
- Added new sprites and animation for all Runes.
- Elements counter can now be clicked to turn it translucent in case it hides any UI elements. (eg. Kaiser Crab fight)
- Added Runes charging animation.
- Adjusted Runes charges meter UI.

### Misc
- Updated BaseLib dependency to 3.2.0
- Updated for beta branch 0.107.0
- Changed when Runes are activated at the start of turn.
  - Cards added by Origo Runes will be before card draw so they'll no longer gets discarded with full hand.
- Flumen Rune will modify start of turn card draw instead of drawing cards directly.
  - As a result, Fiddle will no longer prevent card draw from flumen at turn start (but draw from passive triggers or break will still be blocked.)

### Bug Fixes
- Fixed Duplicate not using up AmpPower when applicable.
